# MBA IA UNIFOR

Este repositório contém os projetos e exercícios desenvolvidos durante o MBA em Inteligência Artificial da UNIFOR.

## Estrutura do Repositório

- `files/`: Contém os arquivos principais do projeto
  - `app.py`: Arquivo principal da aplicação
  - `funcao.py`: Módulo com as funções do sistema
  - `menu.py`: Módulo de interface do usuário
  - `visualizar.py`: Módulo para visualização de dashboard. 

- `datasets/`: Contém os arquivos CSV , para armazenar as despesas e receitas.
  - `depesas.csv`: Arquivo que armazena as despesas
  - `receitas.csv`: Arquivo que armazena as receitas

  
## Personal Organizer

Um sistema de gerenciamento financeiro pessoal que permite:
- Registrar receitas
- Registrar despesas
- Visualizar saldo
- Gerar relatórios

## Como executar

1. Clone o repositório
2. Instale as dependências:
```bash
pip install pandas
```
3. Execute o programa:
```bash
python files/app.py
```
